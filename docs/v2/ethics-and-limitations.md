# Ethics and Limitations

## Non-Diagnostic Positioning
This system is built for educational analysis of public text patterns. It must not be used to diagnose mental health conditions.

## Known Limitations
- Public social-media data is noisy and context-poor
- Platform-specific slang can distort labels
- Multi-label overlap is common and difficult to disambiguate
- A single comment rarely provides enough evidence for strong conclusions

## Required Product Safeguards
- Show uncertainty explicitly
- Suppress strong conclusions on low-volume pages
- Keep crisis-like outputs conservative and reviewable
- Preserve local/private inference when possible
