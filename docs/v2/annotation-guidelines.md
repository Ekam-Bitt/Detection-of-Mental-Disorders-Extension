# Annotation Guidelines

## Goal
Label the language present in a post or comment, not the author.

## Primary Rules
1. Assign one or more signal labels only when the text itself supports them.
2. Use `no_clear_signal` when no supported signal is present.
3. Reserve `crisis_self_harm` for explicit or strongly implied self-harm or suicidal language.
4. Severity reflects the intensity of the language in the text, not the presumed clinical condition.

## Severity Scale
- `0`: none
- `1`: mild
- `2`: moderate
- `3`: high

## Ambiguity Handling
- If evidence is weak, choose the safer low-severity interpretation.
- Add notes whenever sarcasm, quoting, or context collapse affects confidence.
